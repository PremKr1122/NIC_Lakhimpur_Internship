# Fake Voter Detection System using Face Recognition

This project detects duplicate or fake voters from electoral roll PDFs using face recognition and voter ID extraction.

It automatically extracts faces from voter list PDFs, matches them with an existing voter database using deep learning embeddings, and flags potential duplicate voters.

---

## Features

- Face detection using InsightFace
- Face embedding generation
- Duplicate face detection using cosine similarity
- Automatic voter ID extraction from PDFs
- Face-to-voterID mapping
- Automatic dataset creation
- Flask web interface
